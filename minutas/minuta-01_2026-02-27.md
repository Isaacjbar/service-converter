## Acuerdos Tomados

1. Se definió la arquitectura general del proyecto: backend en Django REST Framework y frontend en Vite + React.
2. Se acordó usar JWT para autenticación con refresh token rotation en ambas capas.
3. Se estableció el modelo de usuario personalizado con campo `role` para control de acceso.
4. Se acordó usar `javalang` como librería para el parseo del AST de Java en el backend.
5. Se definió que el frontend manejaría el rendering de diagramas PlantUML mediante compresión `pako` y visualización SVG.
6. Se acordó versionar el proyecto desde el inicio con Git, con ramas separadas por repositorio (`service-converter` / `converter-web-app`).
